# SportsHub Central

Welcome to SportsHub Central, a dynamic and modern web application designed for managing and tracking a college sports festival. This platform provides real-time updates, comprehensive administrative controls, and an engaging user experience for both participants and spectators.

Built with Next.js, Firebase, and Google's Generative AI, SportsHub Central offers a robust solution for bringing your sports event to life online.

## Core Features

-   **Live Match Display**: View real-time scores, statuses (Live, Upcoming, Completed), and detailed information for matches across various sports.
-   **Comprehensive Admin Dashboard**: A powerful interface for administrators to create, update, and delete matches, manage teams, and edit player information.
-   **Dynamic Team and Player Management**: Teams are automatically created when new matches are added, and administrators can manage team rosters with ease.
-   **AI-Powered Match Prediction**: Leverages Google's Gemini model via Genkit to provide intelligent predictions on match outcomes based on team data and live scores.
-   **Secure Authentication**: A simple and secure login system for users and a dedicated admin account with elevated privileges.
-   **Responsive & Themed UI**: A modern, responsive interface built with ShadCN UI and Tailwind CSS, featuring both light and dark modes for a comfortable viewing experience.

## Technology Stack

-   **Frontend**: Next.js (App Router), React, TypeScript
-   **Styling**: Tailwind CSS, ShadCN UI
-   **Backend & Database**: Firebase (Firestore, Authentication)
-   **Generative AI**: Google AI (Gemini) via Genkit
-   **Deployment**: Firebase App Hosting# SportsHub
