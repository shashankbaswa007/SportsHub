# **App Name**: SportsHub Central

## Core Features:

- User Authentication: Secure login/signup with 12-digit username verification.
- Live Match Display: Display real-time scores and match statuses for various sports.
- Sport-Specific Background: Dynamically change the background theme based on the selected sport.
- Match Prediction Tool: AI-powered tool that predicts match outcomes based on live data; determine the likely winner for live/upcoming matches, providing dynamic predictions based on the data and conditions.
- Settings Page: Allow users to change passwords and toggle dark/light mode.
- Real-Time Notifications: Display real-time updates for scores, match statuses, and announcements.

## Style Guidelines:

- Primary color: Electric blue (#7DF9FF) to convey energy and modernity.
- Background color: Dark navy (#191970) to provide contrast and a professional look in dark mode.
- Accent color: Neon yellow (#CFFF29) for highlights and interactive elements.
- Headline font: 'Poppins', sans-serif, bold for a sporty and modern look.
- Body font: 'Inter', sans-serif, for clear and readable text throughout the app.
- Use sport-specific icons for easy navigation and visual appeal.
- Subtle animations for score updates and transitions between sports to enhance user experience.
